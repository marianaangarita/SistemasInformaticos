# Project Brief

## Foundation
El proyecto "Autoescuela de Sistemas" es un simulador de examen interactivo para ayudar a los estudiantes a repasar comandos de Linux/Windows, bases de datos, hardware y contenedores.

## Core Requirements
- Sistema de tests tipo quiz, ya sea online o local.
- Uso obligatorio de la metodología Memory Bank.
- Trabajo colaborativo (control de versiones con Git/GitHub, uso de ramas).
- Modularidad estricta en el código: separación de lógica y presentación.
- Registro estricto de prompts enviados a la IA.

## Goals
Proveer una base de preguntas dinámica y extensible para la evaluación y aprendizaje, garantizando una arquitectura que sirva de ejemplo para otros desarrollos y que cumpla la rúbrica de evaluación.
