# Contexto del Producto

## Visión
Crear un programa de tests (online o local) que funcione como una "Autoescuela de Sistemas", donde los usuarios puedan aprender y ponerse a prueba sobre diversas tecnologías de sistemas: hardware, comandos de Windows y Linux, redes IP, contenedores, entre otros.

## Objetivos
- Proveer un entorno de evaluación interactivo sobre temas de la asignatura de sistemas.
- Construir una aplicación web aplicando la metodología profesional Memory Bank.
- Trabajar en grupo de 2 personas, utilizando obligatoriamente GitHub y ramas (branches) para cada apartado.
