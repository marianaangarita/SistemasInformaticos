# Technical Context

## Technologies Used
- HTML5: Estructura base de la aplicación.
- CSS3 (Vanilla): Estilos modernos, uso de variables nativas y tarjetas de información.
- JavaScript (Vanilla): Lógica de aplicación pura sin frameworks (por requerimiento de simplicidad/rendimiento/encapsulación).
- Git / GitHub: Control de versiones.

## Development Setup
- Servidor local básico (opcional) o apertura directa de archivos estáticos (`index.html`).
- Editor de código estándar (Visual Studio Code o similar).

## Technical Constraints
- `components.js` no puede tener lógica de negocio.
- Solo funciones que devuelvan Strings (HTML).
- Orquestación centralizada en `app.js`.
